## Descripción del Proyecto

El sistema permite:

- Reservas de canchas de fútbol evitando duplicaciones de horarios.
- Venta de productos mediante un e-commerce integrado.
- Registro e inicio de sesión de usuarios.
- CRUD de productos y canchas para administradores.
- Gestión de usuarios (activar, desactivar, eliminar).
- Visualización de canchas, productos y promociones en la página principal.
- Secciones adicionales: Quiénes somos, Contacto, Galería de imágenes, Error 404.

---

## Tecnologías Utilizadas

- **Backend:** Node.js, Express
- **Base de datos:** MongoDB, Mongoose
- **Autenticación:** JWT (JSON Web Token), Bcrypt
- **Manejo de archivos e imágenes:** Multer, Cloudinary
- **Validaciones:** express-validator
- **Otras utilidades:** CORS, Morgan

## Configuración del Backend

**Recomiendo crear las siguiente variables de entorno en el backend.**

-PORT
-MONGODB
-ADMIN_SECRET_KEY
-STAFF_SECRET_KEY
-JWT_SECRET_KEY
-CLOUDINARY_API_SECRET
-CLOUDINARY_API_KEY
-CLOUDINARY_CLOUD_NAME
